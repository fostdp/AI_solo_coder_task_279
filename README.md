# 垃圾分类AR模拟游戏

一个基于网页的垃圾分类AR模拟小游戏，支持手机摄像头和触摸操作。

## 功能特点

- 🎥 **AR摄像头支持**: 使用手机后置摄像头，提供真实的AR体验
- 🎮 **拖拽交互**: 支持鼠标拖拽和触摸屏拖拽操作
- 🗑️ **四种垃圾分类**: 可回收物、厨余垃圾、有害垃圾、其他垃圾
- 📊 **得分系统**: 记录得分、分类次数和正确率
- 💾 **游戏记录**: 支持后端API和本地存储双模式
- ⏸️ **暂停/重新开始**: 完整的游戏控制功能

## 垃圾类型

### 可回收物 (蓝色)
- 🥤 塑料瓶、📦 纸箱、🍶 玻璃瓶、📄 废纸、🥫 易拉罐

### 厨余垃圾 (绿色)
- 🍌 香蕉皮、🍎 苹果核、🥬 菜叶、🍗 骨头、🍚 剩饭

### 有害垃圾 (红色)
- 🔋 电池、💊 药品、💡 灯泡、🧴 杀虫剂、🌡️ 温度计

### 其他垃圾 (灰色)
- 🚬 烟头、🧻 用过的纸巾、🍽️ 一次性餐具、🩹 创可贴、🧹 灰尘

## 运行方式

### 方式一：直接打开（推荐，无需服务器）
直接在浏览器中打开 `index.html` 文件即可游玩。
- 游戏记录会自动保存在浏览器本地存储中
- 支持所有现代浏览器和手机浏览器

### 方式二：使用后端服务器（需要Node.js）
1. 安装依赖：
   ```bash
   npm install
   ```

2. 启动服务器：
   ```bash
   npm start
   ```

3. 在浏览器中访问：`http://localhost:3000`

## 游戏玩法

1. 点击"开始游戏"按钮
2. 允许浏览器访问摄像头（如无法访问，会自动使用模拟背景）
3. 屏幕上会随机出现虚拟垃圾
4. 用鼠标或手指将垃圾拖入对应的垃圾桶
5. 分类正确得10分，错误会提示正确分类
6. 点击"游戏记录"查看历史成绩

## 技术栈

- 前端：HTML5 + CSS3 + 原生JavaScript
- 后端：Node.js + Express
- 存储：JSON文件 / LocalStorage
- API：RESTful API

## 注意事项

- 🔒 摄像头访问需要HTTPS或localhost环境
- 📱 推荐在手机上使用横屏模式获得最佳体验
- 🌐 如无法访问后端服务器，自动降级使用浏览器本地存储
- 💾 游戏记录最多保存20条（服务器）或10条（本地存储）
